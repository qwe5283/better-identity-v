# Better Identity V

BetterIDV · 更好的第五人格， 一个基于计算机视觉技术，意图让第五人格变的更好的项目。

## 功能
* **实时任务**
  * **自动校准**：托管因破译或治疗产生的QTE快速反应事件，自动按下 <kbd>Space</kbd> 处理校准击打。支持后台校准，心跳抬手，自动压机等高级功能。
  * **自动拾取**：检测到可拾取道具时，自动按下拾取到道具栏，适配联合狩猎的双道具栏。支持自定义按键映射，搬运道具等高级功能。
  * **快速购买**：支持设置预购道具，快速打开伯格包或在电话亭买下预购道具。
* **信息指示**
  * **辅助瞄准**：在画面中心添加辅助瞄准线，以便建立屏幕中心感。
  * **冷却可视化**：在画面上显示技能冷却倒计时，以便局内行动决策。
    * 当前支持显示红夫人-水镜、红蝶-刹那生灭、闪现、飞轮效应的冷却倒计时。
* **操作逻辑优化**
  * 优化部分技能的释放逻辑，以便降低上手难度。
* (更多功能及上述部分功能仍在快速迭代与开发中...)
